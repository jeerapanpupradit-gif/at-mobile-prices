# AT Mobile - Dashboard ราคาขายและราคาซ่อมโทรศัพท์มือถือ

เว็บแอปพลิเคชันระบบ Dashboard ราคากลางจัดจำหน่ายโทรศัพท์มือถือและบริการงานซ่อมเครื่อง (AT Mobile Pricing & Repair Dashboard) สำหรับเจ้าของร้านค้า ตัวแทนจำหน่าย หรือช่างซ่อมมืออาชีพ เพื่อความสะดวกในการใช้งาน ค้นหา และคำนวณราคาบริการลูกค้าได้ทันที

## 🚀 วิธีการเข้าใช้งาน (How to Use)
คุณสามารถใช้งานแอปพลิเคชันนี้ได้ทันทีโดยไม่ต้องทำการติดตั้งเซิร์ฟเวอร์หรือคอมไพล์โค้ดใด ๆ:
1. ดาวน์โหลดหรือคัดลอกโฟลเดอร์โครงการนี้ไว้ในคอมพิวเตอร์ของคุณ
2. ดับเบิ้ลคลิกเพื่อเปิดไฟล์ **`index.html`** ด้วยเว็บเบราว์เซอร์ใดก็ได้ (เช่น Google Chrome, Safari, Microsoft Edge, Firefox)
3. ระบบจะทำงานและโหลดข้อมูลจากตัวเก็บข้อมูลภายในเบราว์เซอร์ (`localStorage`) ทันที

---

## 🌟 ฟีเจอร์หลักของระบบ (Core Features)

1. **Dashboard Overview (ภาพรวมร้านค้า)**:
   - สรุปตัวเลขสถิติ (KPIs) สำคัญ: จำนวนรุ่นมือถือทั้งหมดที่มี, ราคาจำหน่ายเฉลี่ยสำหรับเครื่องมือหนึ่ง, ราคาค่าซ่อมเฉลี่ย, และการประเมินแนวโน้มตลาดราคามือถือ
   - กราฟสถิติที่สวยงาม (พัฒนาด้วย Chart.js): แสดงราคากลางเฉลี่ยแยกตามแบรนด์ และอัตราส่วนหมวดหมู่บริการงานซ่อมยอดนิยม

2. **ราคาขายโทรศัพท์ (Phone Selling Prices)**:
   - ค้นหารุ่นและแบรนด์มือถือได้แบบ Real-time
   - ระบบฟิลเตอร์เพื่อคัดกรองเฉพาะแบรนด์ที่ต้องการ (Apple, Samsung, Xiaomi, Oppo, Vivo) หรือฟิลเตอร์ตามความจุที่ระบุ (128GB, 256GB, 512GB, 1TB)
   - แสดงตารางเปรียบเทียบระหว่างราคาเครื่องใหม่แกะกล่อง และราคาเครื่องมือสอง สภาพดี พร้อมสัญลักษณ์แนวโน้มราคากลาง (ราคาขึ้น, ลง, หรือคงที่)

3. **ราคาซ่อม & เครื่องคำนวณ (Repair Prices & Calculator)**:
   - ค้นหาและดูราคางานซ่อมเฉพาะรุ่น (เช่น เปลี่ยนหน้าจอแท้, เปลี่ยนแบตเตอรี่, ซ่อมบอร์ดชาร์จ) พร้อมบอกระยะเวลาซ่อมโดยประมาณและระยะเวลาประกัน
   - **เครื่องมือคำนวณราคา (Repair Billing Calculator)**: สามารถเลือกงานซ่อมหลายรายการพร้อมกัน, ใส่ยอดส่วนลดพิเศษ (Discount), ระบบจะประเมินเวลาซ่อมสะสมทั้งหมด ระยะเวลารับประกันสูงสุด และราคาสุทธิโดยอัตโนมัติ
   - **พิมพ์ใบเสนอราคา (Quotation generator)**: สามารถกดเปิดใบเสนอราคาบริการซ่อมมือถือที่สรุปข้อมูลรุ่น อะไหล่ ราคา และเงื่อนไขการรับประกันเพื่อกดพิมพ์ (Print / PDF) ออกไปให้ลูกค้าได้ทันที

4. **จัดการฐานข้อมูล (Database Admin Settings)**:
   - จัดการข้อมูลหลังร้านผ่านหน้า UI: สามารถเพิ่มรุ่นโทรศัพท์ใหม่, แก้ไขข้อมูลความจุและราคาขาย, ปรับแต่งหรือลบข้อมูลรุ่นที่มีอยู่
   - บันทึกความเปลี่ยนแปลงทั้งหมดอย่างถาวรในเครื่องผู้ใช้ด้วย `localStorage`
   - ปุ่ม **"กู้คืนค่าตั้งต้น"** เพื่อล้างข้อมูลการเปลี่ยนแปลงทั้งหมดและดึงข้อมูลเริ่มต้น (Seed Data) กลับมาใช้งานหากต้องการรีเซ็ต

## 🛠️ โครงสร้างไฟล์ในระบบ (File Structure)
* `index.html` - โครงร่างหน้าเว็บและเลย์เอาต์หลัก รวมถึง Modal ฟอร์ม
* `styles.css` - สไตล์การดีไซน์ระดับพรีเมียม (รองรับ Dark / Light Mode และ Responsive สำหรับมือถือ)
* `data.js` - ข้อมูลจำลองตั้งต้น (Seed Data) และฟังก์ชันติดต่อประสานงานกับ `localStorage`
* `app.js` - โค้ดควบคุมระบบ ควบคุมแถบเมนู ตัวกรอง ค้นหา เครื่องคำนวณ กราฟ และระบบ CRUD หลังร้าน
